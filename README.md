# Puzzle-Game

لعبة ألغاز متوسطة الصعوبة ومريحة للعين، تم تطويرها باستخدام HTML, CSS, وJavaScript.

المميزات
شبكة ألغاز 3x3 متوسطة الصعوبة
عداد الوقت وعدد الحركات
تأثيرات حركة سلسة عند تحريك القطع
رسالة فوز عند إكمال اللغز
تصميم مريح للعين مع ألوان هادئة
طريقة التشغيل
افتح الملف index.html في أي متصفح حديث.
اضغط على زر "ابدأ اللعبة" للبدء.
حاول ترتيب الأرقام حسب الترتيب الصحيح.
